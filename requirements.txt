# 这是项目所有 pip 包的单一来源 (single source of truth).
# environment.yml 通过 `-r requirements.txt` 引用本文件.
#
# 直接用本文件装 (假设 Python 3.10 + PyTorch 2.1.2 + CUDA 11.8 已就位):
#   pip install --no-build-isolation -r requirements.txt
#
# 必须加 --no-build-isolation, 否则 chumpy 的 setup.py 会因为 build subprocess
# 里没有 pip 模块而 fail.
#
# 注意: pytorch3d 单独装, 不在 requirements 里 (PyPI 上没有匹配 wheel).
# 推荐用 Facebook 的预编译 wheel (py310 + cu118 + pyt212):
#   pip install --no-index --no-cache-dir pytorch3d \
#     -f https://dl.fbaipublicfiles.com/pytorch3d/packaging/wheels/py310_cu118_pyt212/download.html
# 或从源码装 (慢, 需要 CUDA toolkit, ~10-30min):
#   pip install --no-build-isolation "git+https://github.com/facebookresearch/pytorch3d.git@stable"
